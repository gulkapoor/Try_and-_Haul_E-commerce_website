# Try and Haul Fashion E-Commerce Web Application

A full-stack fashion e-commerce web application built using **Django** and **PostgreSQL**. The platform enables users to browse fashion products, manage shopping carts, place orders, and securely authenticate through a user-friendly interface.

---
## Features

- User Registration & Login Authentication
- Secure User Sessions using Django Authentication
- Browse Fashion Products
- Product Categories
- Shopping Cart Management
- Place Orders
- Order History
- Admin Panel for Product Management
- Responsive User Interface
- PostgreSQL Database Integration
